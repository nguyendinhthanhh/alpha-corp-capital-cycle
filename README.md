# Alpha Corp Capital Cycle

Dự án website kể chuyện dữ liệu tương tác (Interactive Data Storytelling) phục vụ môn học Kinh tế chính trị Mác - Lênin (MLN122).

Tình huống trung tâm: Điều tra nguyên nhân vì sao tập đoàn bất động sản Alpha Corp vay 10.000 tỷ, xây xong phần thô 3 tòa tháp nhưng vẫn rơi vào tình trạng tê liệt thanh khoản và đứt gãy tuần hoàn tư bản.

## How to run
Dự án sử dụng Vite và React (không cần backend).
```bash
npm install
npm run dev
```
Mở đường dẫn `http://localhost:5173` trên trình duyệt để sử dụng.

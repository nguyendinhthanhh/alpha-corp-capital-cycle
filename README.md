> [!NOTE]  
> **Về Dự án:** Alpha Corp là một siêu ứng dụng Web học tập tương tác (Interactive Web App). Dự án được thiết kế với tham vọng xóa bỏ rào cản hàn lâm của bộ môn Kinh tế Chính trị Mác - Lênin. Thông qua việc nhập vai và phân tích vụ phá sản của tập đoàn bất động sản giả định Alpha Corp, người dùng sẽ tự mình "thấm" được quy luật vận động khốc liệt của dòng vốn ($T \rightarrow H \dots SX \dots H' \rightarrow T'$).

## 🌟 Tầm nhìn Dự án (Vision)

Giáo dục kinh tế học thường mắc kẹt trong những trang giấy đầy chữ và công thức khô khan. **Alpha Corp Capital Cycle Study** ra đời để định nghĩa lại cách chúng ta tiếp cận lý thuyết hàn lâm:
- **Gamification (Trò chơi hóa):** Biến việc học thành việc giải mã một vụ sụp đổ tài chính.
- **Learning by Doing (Học qua thực hành):** Thay vì đọc thuộc lòng khái niệm "Tuần hoàn tư bản", người dùng phải tự tay điều tiết dòng tiền để tránh việc tập đoàn bị vỡ nợ.
- **Contextual Feedback (Phản hồi theo ngữ cảnh):** Mọi lựa chọn sai lầm trong hệ thống đều sẽ trả về một lời giải thích sâu sắc về mặt lý luận.

---

## 🚀 Tính năng Cốt lõi (Core Features)

Dự án là sự kết hợp của 4 phân hệ khổng lồ, tạo thành một hệ sinh thái học tập hoàn chỉnh:

### 1. 📖 Story Mode (Hồ sơ vụ án Alpha Corp)
Một trải nghiệm kể chuyện tương tác (Interactive Data Storytelling) trải dài theo dòng thời gian.
- **Cuộn trang theo dòng sự kiện:** Từ lúc Alpha Corp huy động vốn ($T$), mua vật liệu ($H$), đóng băng dự án ($SX$), đến lúc mất thanh khoản.
- **Lý luận đối chiếu:** Mỗi sự kiện thực tế đều được đính kèm một tab phân tích, vạch trần lỗ hổng lý luận đằng sau quyết định sai lầm của ban lãnh đạo.

### 2. 🧠 The Learning Engine (Dashboard Học tập Cá nhân)
Không chỉ là một trang web tĩnh, hệ thống có một lõi theo dõi học tập (Learning Engine) chạy ngầm:
- **Knowledge Map (Bản đồ tri thức):** Theo dõi % thuần thục của 16+ khái niệm kinh tế lõi (Tư bản tiền tệ, Thời gian lưu thông, Giá trị thặng dư,...).
- **Streak & Activity Tracking:** Hệ thống lưu vết quá trình học tập liên tục để khuyến khích tiến độ.
- **AI Rule-based Tutor:** Dựa trên điểm số trắc nghiệm và lịch sử học, trợ lý ảo tự động sinh ra lời khuyên "Bạn nên ôn phần nào" hoặc "Đã đến lúc làm Quiz khó hơn".

### 3. ⚔️ Quiz Arena (Đấu trường Trắc nghiệm Tuyến tính)
Hệ thống đánh giá năng lực vượt trội:
- **Ngân hàng câu hỏi động:** Gồm hàng chục câu hỏi xoay quanh case study Alpha Corp.
- **Adaptive Grading:** Tính điểm ngay lập tức, chấm điểm dựa trên độ khó.
- **Deep Explanations:** Sau mỗi câu trả lời, popup lý luận sẽ xuất hiện giải thích tại sao đáp án A đúng và B sai dựa trên giáo trình Kinh tế chính trị.

## 🔬 Phòng thí nghiệm Vốn (Capital Lab Simulators)

Đây là **phần kỹ thuật phức tạp nhất** và cũng là tính năng "WOW" nhất của dự án. Hệ thống cung cấp 3 bộ mô phỏng (Simulators) được lập trình bằng các công thức toán học và vật lý thực:

| Tên Giả Lập | Chức năng & Tương tác |
| :--- | :--- |
| **⏳ Temporal Simulator**<br/>*(Giả lập thời gian)* | Cho phép người dùng kéo thanh trượt tốc độ thi công dự án (Thời gian SX) và tốc độ bán hàng (Thời gian LT). Trực quan hóa việc "ngâm vốn" quá lâu sẽ ăn cụt lợi nhuận như thế nào. |
| **🌍 Spatial Simulator**<br/>*(Giả lập không gian)* | Cho phép đứt gãy một chuỗi cung ứng bất kỳ. Quan sát hiệu ứng domino khi tiền tệ không thể biến thành hàng hóa, làm sụp đổ toàn bộ chu kỳ sản xuất tiếp theo. |
| **📉 Crisis Simulator**<br/>*(Giả lập Khủng hoảng)* | Một Mini-game sống còn. Người dùng được cấp một số vốn $T$. Bạn phải liên tục phân bổ vốn vào Vật tư và Sản xuất trong khi thị trường liên tục giáng xuống các cú sốc (Đóng băng tín dụng, Tăng lãi suất). Mục tiêu: Trụ vững 12 tháng không vỡ nợ. |

## ⚡ Kiến trúc Hệ thống (Engineering Architecture)

Dự án là một ví dụ mẫu mực về **Engineering Excellence**, được thiết kế để chịu tải các logic toán học phức tạp của bộ Simulator mà không làm rớt FPS (Frames Per Second) của giao diện.

### 1. Kiến trúc No-Tailwind Custom Design System
Chúng tôi từ chối sử dụng TailwindCSS hay Bootstrap. Dự án sở hữu một Design System được viết hoàn toàn bằng **Vanilla CSS Variables (Tokens)**.
- **Hiệu năng tuyệt đối:** Không tốn chi phí parse class, file CSS siêu nhẹ.
- **Kiểm soát vi mô:** Các hiệu ứng Glassmorphism (kính mờ đa tầng), Dark Mode vũ trụ sâu thẳm, và Fluid Typography (Chữ tự động scale theo màn hình bằng hàm `clamp()`) được căn chỉnh tỉ mỉ tới từng pixel.
- **Scalability:** Mọi màu sắc, khoảng cách (spacing), bo góc (radius) đều gọi qua biến `var(--color-...)`, giúp việc thay đổi theme chỉ tốn 1 dòng code.

### 2. Physics-based UI với Framer Motion
Giao diện không dùng CSS Transitions thông thường mà sử dụng **Physics-based Animations** (spring, damping, mass) của Framer Motion:
- Các Cards khi hover sẽ nổi lên với trọng lực mượt mà.
- Page Transition (Chuyển trang) không bị giật lag, giữ nguyên bối cảnh không gian của người dùng.
- Trực quan hóa dòng tiền chuyển động ($T \rightarrow H$) qua các path SVG được animate realtime.

### 3. Tối ưu hóa State Management (React 19)
Phòng thí nghiệm (Capital Lab) chạy các vòng lặp tính toán lãi suất, dòng tiền và phá sản mỗi mili-giây.
- Để tránh React re-render sập web, chúng tôi tách bạch Local State và Global State.
- Sử dụng các kỹ thuật memoization (`useMemo`, `useCallback`) và Custom Hooks độc lập (`useLearning`, `useSimulator`) để đóng gói logic kinh doanh rời khỏi UI Components.

## 📁 Cấu trúc Mã nguồn Chi tiết (Directory Tree)

Dự án tuân theo chuẩn cấu trúc Feature-based chia theo miền nghiệp vụ (Domain-driven):

```text
alpha-corp-capital-cycle/
├── public/                 # Assets tĩnh (Images, SVGs, Videos)
├── src/
│   ├── components/         # Trái tim của giao diện
│   │   ├── capital-flow/   # Component render mũi tên tuần hoàn tiền
│   │   ├── capital-lab/    # Chứa 3 bộ Simulators vật lý
│   │   ├── hero/           # Màn hình Splash 3D ban đầu
│   │   ├── home/           # Layout trang chủ
│   │   ├── layout/         # Vỏ bọc ứng dụng (Header, Footer kính mờ)
│   │   └── shared/         # Atomic UI (Nút bấm, Alert, Chips)
│   ├── data/               # Kho dữ liệu Mock (Question Banks, Story Lines)
│   ├── learning/           # "Bộ não" của app: Context API, Logic chấm điểm
│   ├── pages/              # Các trang chính (Dashboard, Route, Quiz)
│   ├── App.jsx             # Root Component cấu hình Router v7
│   └── index.css           # 1500+ dòng Custom Design Tokens & Media Queries
├── package.json            # Quản lý dependencies (Rất ít thư viện rác)
└── vite.config.js          # Cấu hình build siêu tốc
```

## 💻 Chạy thử Nhanh (Quick Start)

Môi trường phát triển yêu cầu **Node.js v18+**.

```bash
# 1. Clone mã nguồn dự án về máy
git clone https://github.com/nguyendinhthanhh/alpha-corp-capital-cycle.git

# 2. Truy cập thư mục
cd alpha-corp-capital-cycle

# 3. Cài đặt các gói thư viện (Chỉ mất vài giây do ít thư viện bên thứ 3)
npm install

# 4. Khởi chạy máy chủ phát triển
npm run dev
```

> [!TIP]  
> Sau khi server chạy, nhấn <kbd>o</kbd> + <kbd>Enter</kbd> trong terminal để tự động mở dự án ở địa chỉ `http://localhost:5173`.
